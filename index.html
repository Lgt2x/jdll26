Groupes de travail

DWG

Design Working Group

  • S’assure de la cohérence de l’ergonomie
  • Contribue à l’ergonomie des nouveaux outils
  • Propose des améliorations

CWG

CAD Working Group

  • Apporte une vision professionnelle de la CAO
  • Donne son avis sur les fonctionnalités
  • Présente comment font les autres programmes

CQWG

Code Quality Working Group

  • En cours de formation
  • Donne une ligne directrice sur le code
  • Standardisation des règles de développement

IWG

Infrastructure Working Group

  • S’occupe du site web
  • Gère les différentes plateformes
  • Développe le nouveau site web
